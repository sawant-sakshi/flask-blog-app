# 📝 Flask Blog App

This is a simple blog web application built using **Python** and **Flask** framework.

---

## 🚀 Features

- 🧑‍💻 User Login and Signup
- 📝 Create, Edit, and Delete Blog Posts
- 🖼 Upload Images with Posts
- 🌙 Dark Mode
- 📦 SQLite Database (or easily switchable)

---
